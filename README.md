## 使用教程

### CVC

#### 使用
1. 打开CVC.exe,回车安装证书.
2. 打开你要抓取的作品并重新加载.
3. 回到CVC,复制获取到的`project ID`

#### 退出

回车后等待提出卸载证书的提示,证书删除后自动退出

### CVI

按照提示即可

## 注意事项

1. 若作品加载到99%不动请重启浏览器和CVC再次尝试
2. 请务必正常关闭CVC,否则会导致**无法上网**
